# Cak Suku skills

Claude Code skills I actually use.

## second-opinion

Get a second opinion on a plan, code, design, or decision — the way a
trusted advisor would. Plain, ranked, committed to positions.

### Install

Claude Code, Cursor, Windsurf, Cline, or any agent supporting the Agent Skills standard:

```bash
npx skills@latest add 4RSIM3R/skills --skill second-opinion
```

Or install all skills in this repo:

```bash
npx skills@latest add 4RSIM3R/skills
```

### Use

`/second-opinion`

Or just ask in plain English: "second opinion on this?", "what do you think?", "am I missing anything?"
